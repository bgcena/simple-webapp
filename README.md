# 🚀 Simple Node.js Auth App (Dockerized)

A lightweight authentication app with:

- Node.js (Express)
- MongoDB
- Docker & Docker Compose
- Login / Register functionality

---

## 🧱 Tech Stack

- Express.js
- MongoDB (Mongoose)
- Docker

---

## ⚡ Quick Start

```bash
git clone <this repo>
cd simple-node-auth

cp .env.example .env

docker compose up --build
