# 🚀 Simple Node.js Auth App (Dockerized)

A lightweight authentication app with:

- Node.js (Express)
- MongoDB
- Docker & Docker Compose
- Login / Register functionality

---

## 🧱 Tech Stack

- Express.js
- MongoDB (Mongoose)
- Docker

---

## ⚡ Quick Start

```bash
git clone <your-repo>
cd simple-node-auth

cp .env.example .env

docker compose up --build
```

---

## 🌐 Access

- App: http://localhost:3000
- MongoDB: localhost:27017

---

## 🔐 Features

- User registration
- Password hashing (bcrypt)
- Login validation
- Persistent DB

---

## 🧪 Use Cases

- Auth system demo
- Security testing (WAF / Bot protection)
- API abuse simulation
- Learning Docker + Node
